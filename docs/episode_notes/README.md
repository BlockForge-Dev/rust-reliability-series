# Episode Notes

Create one short markdown file here per published episode.

Suggested format:

- episode number
- failure mode
- what broke
- invariant added
- files changed
- exact Git tag
- follow-up weakness for the next episode

Suggested file names:

- `episode-00-v0-weak-system.md`
- `episode-01-timeouts.md`
- `episode-02-duplicate-execution.md`
